# 19.	System Notifications

In addition to Heartbeat Notifications, RxConsole offers system notifications within the application. These applications are designed to remind or notify users of important information.


| After logging into the RxConsole application, a PDMP Administrator will be able to see the Notification(s) icon in the top right corner under their name. |  |
| --- | --- |
| Clicking on the Notification(s) icon, will display the message relating to the notification, if a notification exists.<br><br>Note: The number of notifications will appear between the “Bell” icon and the word “Notification” on the button. |  |
| Clicking on the blue circle X button will minimize the notification. |  |

Notifications will be shown for:

Heartbeat Notifications

New MOU Worksheets

Scheduled Maintenance Events

Certificate Expirations

Release Updates
