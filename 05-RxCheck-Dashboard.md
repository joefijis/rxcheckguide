# 5.	RxCheck Dashboard

The RxCheck Dashboard provides a location for PDMP administrators and other authorized users to view metrics on their state’s RxCheck usage.

Note: The RxCheck Dashboard will only work after the PDMP SRS is configured. Information regarding the SRS (State Routing Service) including where to download the software can be found in the section titled, State Routing Service (SRS) Configuration.

## 5.1.	Apply filters to the RxCheck Dashboard


| Click the pie graph icon on the left-hand side of the screen to access the RxCheck Dashboard. |  |
| --- | --- |
| Click on the downward-facing arrow for the filter titled “During” to reveal options to filter by. |  |
| Select the period to match the historical timeframe you would like to populate the RxCheck dashboard.<br><br>Note: This filter is set to last 7 days by default. |  |
| Click on the downward-facing arrow for the filter titled “Healthcare Entities (PDMP-Site Code)” to reveal a list of healthcare entities. |  |
| Select the desired healthcare entity(ies) by:<br>Scrolling through the dropdown and checking the box next to each desired entity, or<br>Typing in the Search Bar to filter the entities.<br><br>Note: By default, all options are selected. To clear your selections, click the checkbox next to the search bar before choosing your desired options. |  |
| Click on the downward-facing arrow for the filter titled “PDMP Sites” to reveal a list of states. |  |
| Select the desired PDMP Site(s) by:<br>Scrolling through the dropdown and checking the box next to each desired state, or<br>Typing in the Search Bar to filter the states.<br><br>Note: By default, all options are selected. To clear your selections, click the checkbox next to the search bar before choosing your desired options. |  |
| Uncheck the “Include Integrations” box to exclude transaction counts from out-of-state healthcare entities.<br><br>Note: This option is selected by default. |  |
| Click on the downward-facing arrow for the filter titled “Status Types” to reveal a list of status options. |  |
| Select the desired Status Type(s) by:<br>Scrolling through the dropdown and checking the box next to each desired status, or<br>Typing in the Search Bar to filter the status types.<br><br>Note: By default, all options are selected. To clear your selections, click the checkbox next to the search bar before choosing your desired options. |  |
| Click on the Apply Filter button to apply your selected filters on the dashboard. |  |

## 5.2.	View incoming and outgoing requests in the RxCheck Dashboard


| Click on either the Incoming or Outgoing button located below the filters on the RxCheck Dashboard screen. This will allow you to view the selected requests.<br><br>Note: The Incoming option is selected and displayed by default. |  |
| --- | --- |
| Perform the steps from the “Apply filters to the RxCheck Dashboard” section of this guide. |  |
| Select the View by Status button to view a graphical representation of the data, based on the status. |  |
|  |  |
| Select the View by Role button to view the total count of requests made by professionals in a certain role. |  |
